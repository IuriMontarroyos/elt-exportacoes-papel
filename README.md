## Objective
Process export data of paper packaging products, aggregating total exported tons by country and packaging type.

## Technologies
- Python
- Pandas
- CSV files
- Google Colab

## ETL Steps
- **Extract:** Read export data from a CSV file
- **Transform:** Clean and aggregate the data
- **Load:** Generate a final report as a CSV file

## Output
The pipeline generates a summarized report with total exported tons per country and packaging type.
